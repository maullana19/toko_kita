1. Download Git yang berisi paket Gitbash masing-masing
2. Disarankan untuk mendownload NodeJs Karena ada proses flutter menggunakan NPM
3. cek Dependencies flutter, Lalu update jika belum diupdate, update juga bahasa dart
4. GitClone dari repository git kelompok 1 
5. buka file yang sudah diclone yaitu toko_kita lewat VSC lalu cek Panduan Ini
6. Semua codingan merupakan beberapa code berdasarkan modul MobilePrograming
7. cek folder lib, buka file main.dart
    ^main.dart dijadikan App program pertama yang dijalankan

8. Silahkan pilih proses running masing-masing bisa menggunakan Emulator, Browser, atau semua perangkat kesukaan kalian.
9. Proses select device ketik "CTRL + P", Lalu pilih devicenya
10. Selanjutnya running file main.dart nya, Bisa tekan F5 atau pilih manual berdasarkan debug console atau tanpa debug.
11. Jangan lupa nyalakan XAMPP localhostnya Untuk REST API 
12. Jika terkendala atau ada codingan yang tak sesuai bisa komunikasikan karena projectnya belum sempurna.
13. Dibutuhkan Koneksi internet yang cepat
